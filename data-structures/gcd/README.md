
## GCD

- Greatest common divisor.
- Greatest factor that divides both A and B.

### Examples

- GCD(15,25) = 5
- GCD(12,30) = 6
- GCD(10,-25) = 5

### Properties

- GCD(A,B) = GCD(B,A)
- GCD(A,B) = GCD(|A|, |B|)
- GCD(0,A) = |A|
- GCD(A,B,C) = GCD(A, GCD(B,C)) = GCD(B, GCD(A,C)) = GCD(C, GCD(A,B))
- GCD(A,B) = GCD(A-B,B) Given: A>=B
- GCD(A,B) = GCD(A%B,B) Given: A>=B
